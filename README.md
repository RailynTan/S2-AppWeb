# S2 AppWeb
 
 Para la correcta ejecucion del programa se debe de tener instalado 
 tanto Node.js como npm

 Tras esto, ademas de haber descargad y extraido el proyecto
 en una consola (cmd), mediante el comando cd y copiando y pegando
 la ruta de la carpeta del proyecto se deberia de ejecutar el comando
 npm install
 que instalar las dependencias necesarias para ejecutar el proyecto, tras eso se debera de ejecutar el comando
  npm run dev
para la ejecucion del proyecto, tras esto la consola mostrara una direccin web que se debe de ingresar en el navegador
